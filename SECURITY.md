# Security Policy

## Supported versions

Security updates are applied to the latest release branch.

## Reporting a vulnerability

Please do not report security issues in public GitHub issues.

Email **security@mantler.ai** with:

- a clear description of the vulnerability
- reproduction steps or proof-of-concept
- impact assessment (what could be affected)
- any proposed mitigation ideas (optional)

We aim to acknowledge reports within 48 hours and provide a fix timeline for confirmed issues.
